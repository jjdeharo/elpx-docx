elpx-docx

Conversión de proyectos .elpx a .docx reutilizando el exportador HTML5 Single Page de eXeLearning.

Este repositorio contiene una aplicación web en Bun. La conversión real necesita ejecutar el backend y el CLI de eXeLearning, así que GitHub Pages solo puede mostrar esta portada estática, no el conversor funcionando.

Limitación importante: GitHub Pages no puede ejecutar Bun, ni lanzar el exportador de eXeLearning, ni generar el .docx. Para usar la aplicación hay que arrancarla en un servidor o en local.

Ver repositorio Leer README
Arranque local: bun install y bun run start. La interfaz real queda disponible en http://localhost:3007.
